The Dogecoin Wallet

The most advanced, secure, and feature-rich Dogecoin wallet. True protocol implementation with 3-6 second payment detection and incredibly low 0.01 DOGE per KB fees. Includes family mode for kids and family members, automatic recurring payments, digital signature support for messages and files, Write a Check feature for time-locked payments using P2SH CLTV addresses (perfect for rent, gifts, and scheduled payments, just like traditional bank checks), RadioDoge to send signed Dogecoin transactions without internet, Point of Sale (POS) system for merchants with web-based storefront and inventory management, Payment Terminal Mode for secure PIN-protected POS systems with kiosk mode, comprehensive accounting reports with CSV/JSON/PDF export, DogeConnect protocol support for secure encrypted payment envelopes, Atomic Swap for trustless cross-chain exchanges (DOGE ‚Üî BTC/LTC), and Exchange integration for buying and selling Dogecoin directly from your wallet. Built with love for the Dogecoin community. Such features, very wow!

Exchange (Labs)

Buy and sell Dogecoin directly from your wallet using Metal Pay Connect integration. Seamlessly convert between Dogecoin and fiat currencies with secure API integration. Supports multiple fiat currencies and automatically integrates with your wallet address and balance.

Atomic Swap (Labs)

Exchange Dogecoin for Bitcoin or Litecoin using trustless cross-chain atomic swaps. No centralized exchanges needed - swap directly peer-to-peer using Hash Time Lock Contracts (HTLC) for secure, decentralized trading.

Learn More

Recurring Payments

Schedule one-time or monthly recurring payments with precise date/time control and OP_RETURN support.

Learn More

Payment Terminal Mode

Secure PIN-protected payment terminal for POS systems with full-screen kiosk mode. Perfect for merchants and businesses with QR code display and payment monitoring. Exit requires PIN authentication.

Learn More

Point of Sale (POS)

Complete e-commerce solution for merchants. Manage products, categories, and inventory. Access your storefront from any browser on your local network. Real-time payment detection with QR code payments.

Learn More

Write a Check

Create time-locked Dogecoin payments just like traditional bank checks! Uses P2SH CLTV addresses that lock funds until a specified date at the blockchain level. Includes expiration dates, automatic cancellation, status tracking, and PDF generation. Perfect for rent payments, gifts, scheduled payments, and more.

Learn More

Use Doge

Discover stores and businesses that accept Dogecoin near you! Interactive GPS-enabled map with store listings, filtering, and community-contributed sources. Add your store or contribute to the community database.

Learn More

RadioDoge

Send signed Dogecoin transactions without internet using LoRa devices! Connect with KeyChain LoRa or any LoRa-enabled device for offline transactions. Powered by RadioDoge V3 from Dogecoin Foundation.

Learn More

Blockchain Sync Checkpoints

Reset blockchain synchronization with custom checkpoints for faster initial sync. Select from predefined checkpoints to start sync from a specific block height instead of the beginning.

Learn More

Custom Transaction Fees

Configure your default transaction fee category (Economic, Normal, Priority) in Settings. The selected fee category is automatically applied to all transactions including sending coins, sweeping wallets, and canceling checks.

Learn More

DogeConnect

Secure payment protocol for processing encrypted payment envelopes. Compatible with DogeConnect specification, allowing your wallet to parse and process secure payment requests from merchants and services. From Dogecoin Foundation.

Accounting Reports
