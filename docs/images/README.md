# Dogecoin Wallet Website Images

This folder contains all the images and graphics for the Dogecoin Wallet website.

## Created Images

### Icons & Logos
- `dogecoin-logo.svg` - Main Dogecoin logo with gradient styling
- `favicon.svg` - Website favicon
- `github-icon.svg` - GitHub icon for source code links
- `twitter-icon.svg` - Twitter icon for social media links

### Store Badges
- `google-play-badge.svg` - Google Play Store download badge
- `apple-store-icon.svg` - Apple Store icon for iOS version

### App Screenshots
- `hero-phone.svg` - Main hero section phone mockup with wallet interface
- `screenshot-1.svg` - Main wallet screen showing balance and transactions
- `screenshot-2.svg` - Send Dogecoin interface with QR scanning
- `screenshot-3.svg` - Settings screen with all features
- `wallet-hero.svg` - QR code generation screen

## Image Specifications

All images are created as SVG files for:
- Scalability across all screen sizes
- Small file sizes
- Crisp rendering on high-DPI displays
- Easy customization and theming

## Usage

These images are referenced in the HTML files and styled with CSS for consistent appearance across the website.
